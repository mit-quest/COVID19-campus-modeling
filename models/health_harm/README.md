# Health harm models

This folder contains all the `health_harm` models that are used in the Covid-19 health management system.

This is the model which takes in the infection and person samples and simulates people's COVID-19 outcomes: did they go to the hospital, etc.

Currently, the `health_harm_demo` model is a placeholder model.
